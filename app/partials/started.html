Using the Grade Visualizer

Adding Courses

You can either add courses 1 at a time, or use the bulk insert to add many at once.

Single Course

On the Profile Detail page:

  1. click + New Course
  2. Fill in the form, every field is required
  3. Click the green button labelled + New Course
You can cancel the process at any time clicking the red button label cancel.
You can see how this course has changed your grades on the User Dashboard page.

Bulk Course Insert

Bulk inserting is also very simple.

  1. Go to your brock Course and Mark inquiry page.
  2. Highlight all the courses you wish to insert
  3. Copy them to your clipboard with Ctrl + c
  4. On the Profile Detail page, paste the contents of your clipboard with Ctrl + p into the input field labelled + Bulk Insert
  5. Click the button labelled insert
Your courses are now all inserted. You can see how these courses have changed your grades on the User Dashboard page.

Removing Courses

To remove a course:

  1. Go to the Profile Detail page
  2. Search for the course you wish to delete or scroll through the list to find it
  3. Click the eraser in the last column of the courses row
